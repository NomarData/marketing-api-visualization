Very quick Readme
=================

This project is completely static, pure javascript html.
Just copy and paste in any http server and you are done.


The test suite uses Mocha + Unit.js and runs over Node.js
---------------------------------------------------------

1. **To install it:**

```
npm install
```
2. **To run tests:**
```
npm test
```
